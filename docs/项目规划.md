# 公司内容推送工作台规划

## 1. 目标

把每周/每日内容推送从“手工改 HTML”升级为“傻瓜式填充 + 一键获取 + 自动排版 + 可发布 URL”。

## 2. 使用对象

- 非技术同事
- 内容编辑
- 运营同学

## 3. 核心场景

### 3.1 AI 科技周报

- 固定网站采集
- LLM 自动整理
- 人工勾选
- 自动生成 HTML 和企微推送内容

### 3.2 每日物流快讯

- 战争
- 极端天气
- 港口/航线/清关变化
- 影响跨境物流的风险信息

### 3.3 汇率快讯

- 美元、人民币、欧元等汇率变化
- 对报价和成本的影响

### 3.4 知识库快讯

- 最新物流规则
- 操作注意事项
- 客服可直接引用的问答内容

## 4. 产品原则

- 傻瓜式操作
- 固定入口
- 固定模板
- 固定来源
- 先人工确认，再发布
- Windows 和 Mac 都能用

## 5. 工作流

1. 选择推送类型
2. 点击一键获取内容
3. 系统抓取固定网站
4. LLM 整理成候选条目
5. 人工勾选和微调
6. 自动排版成 HTML
7. 导出 HTML 或生成 URL
8. 生成企微推送内容

### 5.1 抓取与候选池规则

- 每个来源单次最多抓取 5 条
- 每个模块候选池最多保留 7 条热点内容
- 候选池优先保留以下类型：
  - 关税政策
  - 战争冲突
  - 禁运/制裁
  - 海峡/港口异常
  - 航线中断
  - 极端天气
  - 运价剧烈波动
- 候选池里的内容需要人工审核后，才能加入正式周报/快讯条目

## 6. 模块设计

### 6.1 表头设置

- Logo
- 期号
- 精选数量
- 主标题
- 日期
- 主题标签

### 6.2 条目编辑

- 标题
- 标签
- 来源链接
- 图片
- 正文

### 6.3 LLM 处理

- 摘要压缩
- 标题改写
- 标签生成
- 风险判断
- 图片说明生成

### 6.4 候选池

- 抓取回来的内容先进入候选池，不直接入稿
- 每条候选内容展示：
  - 标题
  - 来源
  - 发布时间
  - 风险等级/热点等级
  - 摘要
- 人工可执行操作：
  - 勾选加入周报
  - 删除不需要的候选项
  - 调整顺序
  - 按模块重新筛选
  - 只看高热点/高风险内容

### 6.5 推送输出

- HTML 文件
- URL
- 企业微信 JSON
- 填写清单
