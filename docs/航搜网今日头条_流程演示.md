# 航搜网今日头条流程演示

## 说明

这是一版基于用户提供截图的手工模拟流程，用来验证“抓取 -> 候选池 -> 人工审核 -> 入稿”的产品逻辑。

当前限制：

- 还没有真实网页 URL
- 还没有自动抓取器
- 本演示使用截图中可见标题作为候选内容来源

## 一、来源页面

页面来源：航搜网 `今日头条`

页面栏目：

- 物流前沿
- 外贸时讯
- 跨境快报
- 运价趋势

## 二、抓取规则

- 每个来源最多抓取 5 条
- 每个模块候选池最多保留 7 条
- 优先保留：
  - 关税政策
  - 战争冲突
  - 禁运/制裁
  - 海峡封锁
  - 港口异常
  - 航线中断
  - 极端天气
  - 运价波动

## 五、人工审核建议

### 建议保留

- 关税政策类
- 改港/暂存/超期收费类
- 直接影响海运履约和成本的内容

### 建议暂缓

- 马士基重返办公室
- 首批中国制造电动车登陆北美
- 空箱比例新闻

这些内容热度可能有，但对当前跨境物流履约、报价、关税、时效的直接影响弱一些。

## 六、建议入稿的 3 条

### 1. 美国301调查结果公布：拟加征12.5%关税

- 类型：关税政策
- 风险等级：高
- 影响方向：成本上升、报价波动、客户沟通压力增加

### 2. 赫伯罗特最新通知：这批货物改港暂存，超期收费

- 类型：港口/航线异常
- 风险等级：高
- 影响方向：改港、仓储、滞留、超期费用

### 3. 欧盟对华这类产品加税42.3%

- 类型：关税政策
- 风险等级：高
- 影响方向：出口成本、市场策略、清关与报价
